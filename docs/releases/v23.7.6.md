# ISCY V23.7.6

ISCY V23.7.6 baut den Rust-only-Betrieb als echte Monitoring- und Betriebszentrale aus. Der Release ergaenzt die Prometheus-Ausgabe um eine kopierbare Scrape-Konfiguration direkt auf der Statusseite und dokumentiert den Betrieb fuer Prometheus, Grafana, Alerts und agentenlesbare JSON-Auswertungen.

## Highlights

- Prometheus-Scrape-Konfiguration direkt unter `/status/` kopierbar.
- Neue Operations-Monitoring-Doku unter `docs/OPERATIONS_MONITORING.md`.
- README und Handbuch verlinken jetzt die Monitoring-Doku und die relevanten Betriebsendpunkte.
- Rust Backend Version auf `0.3.2` angehoben.
- NVD-Import bleibt async und kann fuer lokale oder air-gapped Tests `file://`-JSON-Fixtures lesen.

## Betriebsendpunkte

- `/health/live`
- `/health/ready`
- `/status/operations.json`
- `/api/v1/status/operations`
- `/metrics`
- `/api/v1/status/metrics`

## Verifikation

- `cargo test --manifest-path rust/iscy-backend/Cargo.toml`
- `make rust-smoke`
- `nix flake check`

## Start auf NixOS

```bash
./start.sh
```

Danach ist ISCY unter `http://127.0.0.1:9000/login/` erreichbar.
