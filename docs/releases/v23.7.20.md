# ISCY V23.7.20

ISCY V23.7.20 ergaenzt das Management-Review- und Audit-Paket als neuen Rust-only Steuerungsworkflow. Damit kann ein Tenant aus laufenden Daten ein auditierbares Review-Paket erzeugen, pruefen und freigeben.

## Neu

- Neue Weboberflaeche `/management-reviews/` fuer Management-Review-Pakete.
- Neue API-Endpunkte:
  - `GET /api/v1/reports/management-reviews`
  - `POST /api/v1/reports/management-reviews`
  - `GET /api/v1/reports/management-reviews/{review_id}`
  - `PATCH /api/v1/reports/management-reviews/{review_id}`
- Migration `0019_rust_management_review_packages` mit persistierter Review-Tabelle.
- Demo-Seed fuer ein Management-Review-Paket.
- Automatisch erzeugte Snapshots mit Kennzahlen, Top-Risiken, ISCY-27-Control-Gaps, Evidence-Luecken, Incident-Entscheidungen, Roadmap-Fokus, Product-Security-Lage und Agent-Posture.
- Review-Status mit Draft, In Review, Approved und Archived sowie Entscheidung, naechsten Massnahmen, Freigabe-User und Freigabezeitpunkt.

## Dokumentation

- README, Handbuch, strategische Roadmap, Operations-, GUI- und Agent-Doku wurden auf V23.7.20 / Rust 0.3.16 aktualisiert.
- Das PDF-Handbuch wurde aus der aktuellen Markdown-Doku neu erzeugt.
