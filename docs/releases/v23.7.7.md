# ISCY V23.7.7

ISCY V23.7.7 ergaenzt den Rust-only-Betrieb um konkrete Monitoring-Artefakte fuer Prometheus, Alertmanager und Grafana. Die Statusseite wurde zur kompakten Betriebszentrale erweitert und zeigt nun neben Health, Migrationen, Modulen, offenen Signalen und Scrape-Konfiguration auch einen Grafana-Query-Spickzettel.

## Neu

- Grafana-Dashboard als importierbare JSON-Datei unter `deploy/monitoring/grafana/iscy-operations-dashboard.json`.
- Prometheus-Scrape-Beispiel unter `deploy/monitoring/prometheus/iscy-scrape.yml`.
- Prometheus-Alert-Regeln unter `deploy/monitoring/prometheus/iscy-operations-alerts.yml`.
- Alertmanager-Routing-Beispiel unter `deploy/monitoring/alertmanager/iscy-alertmanager.yml`.
- Statusseite `/status/` um Grafana-Query-Spickzettel erweitert.
- Operations-Doku, README und Handbuch auf Monitoring-Deploy-Artefakte aktualisiert.
- Rust Backend Version auf `0.3.3` angehoben.

## Betrieb

Start lokal auf NixOS:

```bash
./start.sh
```

Monitoring-Pruefung:

```bash
curl -fsS http://127.0.0.1:9000/status/operations.json
curl -fsS http://127.0.0.1:9000/metrics
```

Grafana-Import:

```text
Dashboards -> New -> Import -> Upload JSON file
deploy/monitoring/grafana/iscy-operations-dashboard.json
```

Die Webhook-URLs im Alertmanager-Beispiel sind Platzhalter und muessen im produktiven Betrieb durch eigene Receiver ersetzt werden.
