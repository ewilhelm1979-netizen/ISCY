# ISCY V23.7.12

ISCY V23.7.12 macht Alertmanager-Fallakten betrieblich ruhiger: wiederholte firing Alerts erzeugen keine neuen Incident-Duplikate mehr, sondern werden anhand des Alertmanager-Fingerprints oder ersatzweise des Alertnamens an die offene Fallakte gehaengt.

## Aenderungen

- Alertmanager-Persistenz dedupliziert offene Incidents ueber `authority_reference`.
- Bei Deduplizierung entsteht eine Timeline-Notiz in der bestehenden Fallakte.
- Webhook-Response meldet `deduplicated_incidents`.
- Statusseite `/status/` verlinkt direkt auf Incident-Fallakten.
- Grafana-Panel `Alert Incidents` enthaelt einen Incident-Drilldown-Link.
- README, Operations-Doku, Handbuch, Screenshot-Doku und Agent-Doku sind auf `V23.7.12 / Rust 0.3.8` aktualisiert.
